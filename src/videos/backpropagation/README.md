# Backpropagation

Motion Canvas implementation for the Markdown-first backpropagation video.

## Current implementation

Only Act I is implemented. The scene follows the source files in `content/videos/backpropagation/`, especially `04-animation-spec.md`, and introduces the perceptron, weights, a linear decision boundary, and the limitation that one line cannot separate every pattern.

## Run locally

```bash
npm run start:backpropagation
npm run build:backpropagation
npm run screenshots:backpropagation
```

Generated screenshot previews are written to `artifacts/screenshots/backpropagation/` and should not be committed.
